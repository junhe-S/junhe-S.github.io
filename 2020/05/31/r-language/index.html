Applied Time Series Analysis (ATSA) with R

Author: John Ho Date: May 31, 2020  11:13:21 Category: 学习

Stochastic Process

A collection of random variables

$$\lbrace X(t):t\in T \rbrace$$

where T is an index set

Other names:

  • times series  (if T is time)
      called continuous or discrete based on the index set
  • random function

Typical Notation

  • $X(t)$ denotes continuous time series
  • $X_t$ denotes discrete time series

  • Random variable $X$

    • function whose value depends on $\omega \in \Omega$
    • observed value, $x$, of the $r.v. X$ is the value of this function at the observed $\omega \lbrace i.e. x = X(w)\rbrace$

  • Realization $x(t)$

    • the set of real-valued outcomes $\lbrace x_t ; t \in T\rbrace$ for a fixed value of $w$ in the sample space, $i.e. x_t = X_t(w)$
    Example
    • $$X(t)=sin(.4t+Z), t\in (-\infty, \infty)$$

      where $Z$ ~ $U[0,2\pi]$

        Selecting $\omega$ gives $z=Z(\omega)$, and letting $t\in (0,60)$
        then the resulting realization is $x(t)=sin(.4t+z), t\in(0,60)$

Appendix

    Ensemble

    • totality of all realizations

    Mean of $X_t$

    • ensemble average of process at time t

      • $$E[X_t]=\mu_t$$

    Variance of $X_t$

      $$var[X_t]=\sigma_t^2$$

    Autocovariance function

      $$cov [ X_{t_1} , X_{t_2}]=E[(X_{t_1} - \mu_{t_1})(X_{t_2} - \mu_{t_2})]= \gamma (t_1 , t_2)$$

    Autocorrelation function

      $$\rho (t_1 , t_2)=\frac{\gamma (t_1 , t_2)}{\sigma_{t_1}\sigma_{t_2}}$$
