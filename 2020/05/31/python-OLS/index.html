python_OLS

Author: John Ho Date: May 31, 2020  23:34:34 Category: 学习

statsmodels.OLS (endog, exog, missing, hasconst) $$Y = \beta_0 X_0 + \beta_1 X_1 + \dots +\beta_n X_n $$

  • endog (endogenous variable)，是上面模型中的 $Y(t)$, 输入是一个长度为 k 的 array。
  • exog (exogenous variable), 即模型中的$\beta_0 X_0 + \beta_1 X_1 + \dots +\beta_n X_n $。
    • 假设回归公式是: $$Y = 1 + 0.1X_1 + 10X_1^2$$ 每次运行代码时设置相同的seed，则每次生成的随机数也相同，如果不设置seed，则每次生成的随机数都会不一样。 
    np.random.seed(9876789)
    我们从最简单的一元模型开始，虚构一组数据。首先设定数据量，也就是上面的 k 值。 
    nsample = 100
    然后创建一个 array，是上面的 $x_1$ 的数据。这里，我们想要 $x_1$ 的值从 0 到 10 等差排列。 
    x = np.linspace(0, 10, nsample)
    转化数组为矩阵, 额外添加了另一个$x_2$ 。而$x_2 = {x_1^2}$ 
    X = np.column_stack((x, x**2))
    使用 sm.add_constant() 在 array 上加入一列常项1。 
    X = sm.add_constant(x)
    然后设置模型里的 $\beta_0$,$\beta_1$，这里要设置成 1,0.1,10。 
    beta = np.array([1, 0.1, 10])
    然后还要在数据中加上误差项，所以生成一个长度为k的正态分布样本。 
    e = np.random.normal(size=nsample)
    由此，我们"点乘"，生成反应项 $Y(t)$。 
    y = np.dot(X, beta) + e
    好嘞，在反应变量和回归变量上使用 OLS() 函数。 
    model = sm.OLS(y,X)
    然后获取拟合结果。 
    results = model.fit()
    再调取计算出的回归系数。 
    print(results.params)
    得到 
    [ 1.34233516 -0.04024948 10.01025357]
    和实际的回归系数非常接近。 
    [1, 0.1, 10]
    当然，也可以将回归拟合的摘要全部打印出来。 
    print(results.summary())

    我们还可以将拟合结果画出来。先调用拟合结果的 fittedvalues 得到拟合的 y 值。 
    y_fitted = results.fittedvalues
    然后使用 matplotlib.pyploft 画图。首先设定图轴，图片大小为 8×6。 
    fig, ax = plt.subplots(figsize=(8,6))
    画出原数据，图像为圆点，默认颜色为蓝。 
    ax.plot(x, y, 'o', label='data')
    画出拟合数据，图像为红色带点间断线。 
    ax.plot(x, y_fitted, 'r--.',label='OLS')
    加入改变坐标轴区间的指令 
    ax.axis((-0.05, 2, -1, 25))

    整体源代码：

    import numpy as np
import statsmodels.api as sm
import matplotlib.pyplot as plt
from statsmodels.sandbox.regression.predstd import wls_prediction_std

np.random.seed(9876789)
# 每次运行代码时设置相同的seed，则每次生成的随机数也相同，如果不设置seed，则每次生成的随机数都会不一样。

'''OLS estimation'''

nsample = 100
x = np.linspace(0, 10, nsample)  # 数组：等差数列
X = np.column_stack((x, x**2))  # 转化数组为矩阵
beta = np.array([1, 0.1, 10])  # 1*3矩阵
X = sm.add_constant(X)  # 在 array 上加入一列常项1 注意第一列
e = np.random.normal(size=nsample)  # 误差项：随机抽取100个服从"标准正态分布"的样本
y = np.dot(X, beta) + e  # 生成反应项 y(t)
model = sm.OLS(y, X)  # OLS模型
results = model.fit()  # 模型拟合
print('Parameters: ', results.params)  # 回归系数
print(results.summary())  # 摘要
print('R2: ', results.rsquared)。
# 绘图
fig, ax = plt.subplots(figsize=(8,6))
y_fitted = results.fittedvalues
ax.plot(x, y, 'o', label="data")
ax.plot(x, y_fitted, 'r--.',label='OLS')
plt.show()
