python-OLS_dummy_variables

Author: John Ho Date: June 1, 2020  19:58:07 Category: 学习

statsmodels.api.categorical(data, col=None, dictnames=False, drop=False) $$Y = \beta_0 X_0 + \beta_1 X_1 + \dots +\beta_n X_n $$

  • endog (endogenous variable)，是上面模型中的 $Y(t)$, 输入是一个长度为 k 的 array。
  • exog (exogenous variable), 即模型中的$\beta_0 X_0 + \beta_1 X_1 + \dots +\beta_n X_n $。
    • 假设一个反应变量 $Y$ 对应连续自变量 $X$ 和一个分类变量 $Z$。其中 $Z_i$ 取值于 {0,1} $$Y = 10 + X + 3Z_1 + (-3)Z_2$$ 设定数据量，也就是上面的 k 值。

    整体源代码：

    import numpy as np
import statsmodels.api as sm
import matplotlib.pyplot as plt
from statsmodels.sandbox.regression.predstd import wls_prediction_std

np.random.seed(9876789)
'''OLS with dummy variables'''
nsample = 50
groups = np.zeros(nsample, int)
groups[20:40] = 1
groups[40:] = 2
# dummy = (groups[:,None] == np.unique(groups)).astype(float)

dummy = sm.categorical(groups, drop=True)
x = np.linspace(0, 20, nsample)
# drop reference category
X = np.column_stack((x, dummy[:,1:]))
X = sm.add_constant(X, prepend=False)

beta = [1., 3, -3, 10]
y_true = np.dot(X, beta)
e = np.random.normal(size=nsample)
y = y_true + e

print(X[:5,:])
print(y[:5])
print(groups)
print(dummy[:5,:])

res2 = sm.OLS(y, X).fit()
print(res2.summary())

'''Draw a plot to compare the true relationship to OLS predictions:'''
prstd, iv_l, iv_u = wls_prediction_std(res2)

fig, ax = plt.subplots(figsize=(8,6))

ax.plot(x, y, 'o', label="Data")
ax.plot(x, y_true, 'b-', label="True")
ax.plot(x, res2.fittedvalues, 'r--.', label="Predicted")
ax.plot(x, iv_u, 'r--')
ax.plot(x, iv_l, 'r--')
legend = ax.legend(loc="best")
plt.show()
